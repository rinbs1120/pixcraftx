# ColorForge

AI-powered coloring page generator that creates printable coloring pages in seconds.

## Tech Stack
- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS 4
- Clerk (Auth)
- Supabase (Database)
- Fal.ai (AI Image Generation)

## Getting Started

```bash
pnpm install
cp .env.example .env.local
pnpm dev
```

## Environment Variables

See `.env.example` for required environment variables.
